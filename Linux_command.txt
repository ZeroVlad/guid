Виключити/перезапустити систему: 
halt/reboot

Команди для роботи з файлами:
1. ls - друкує вміст каталогу
2. cat - друкує вміст файлу
3. cd - дозволяє перейти з поточного каталогу в зазначений
4. pwd - друкує на екран поточний каталог
5. mkdir - створення нових каталогів
6. file - показує тип файлу. 
7. cp - копіювання файлів і каталогів
8. mv - переміщення або перейменування файлів і каталогів
9. rm - видаляє файли і папки
10. ln - створює символічні або жорсткі посилання на файли
11. chmod - змінює права доступу до файлу
12. chown - змінює власника файлу
13. find - пошук в файлової системі, файлах і папках. 
14. locate - веде пошук в базі даних updatedb для шаблонів імен файлів
15. du - показує розмір файлу або каталогу
16. df - аналізатор дискового простору
18. mount / umount - команди для підключення і відключення файлових систем Linux
19. wget - скачати з інтернету за посиланням

Команди для роботи з текстом
1. less - перегляд довго тексту
2. head / tail - виводить кілька перших/останніх рядків з файлу
3. grep - шукає текст за шаблоном. 
4. sort - сортування рядків тексту за різними критеріями
5. wc - рахує кількість рядків, слів, байт і виводить ім'я файлу
6. diff - показує відмінності між двома файлами в порядковому порівнянні
7. echo - виводить набраний тест в консолі

Команди для керування процесами
1. kill / xkill / pkill / killall - служать для завершення процесів
2. ps / pgrep - ps виводить інформацію про запущені процеси(або про конкретний). Pgrep працює наступним чином: ви задаєте ім'я процесу, а утиліта показує його ідентифікатор.
3. htop - перегляд запущених процесів в режимі реального часу
4. time - час виконання процесу

Команди оточення користувача
1. su / sudo - запустити програму від імені іншого користувача. su перемикає вас на іншого користувача, а sudo тільки виконує команду від його імені
2. date - виводить дату і час в стандартний висновок. 
3. alias - команда створює синоніми для інших команд Linux
4. uname - виводить основну інформацію про систему
5. uptime - повідомляє вам час роботи системи

Команди для керування користувачами
1. useradd / userdell / usermod - дозволяють додавати / видаляти / змінювати облікові записи користувачів
2. passwd - ця команда дозволяє змінити пароль облікового запису користувача

Команди для перегляду документації
1. man / whatis - команда man відкриває керівництво по певній команді. Whatis показує, які розділи посібників є для даної команди
2. whereis - показує повний шлях до виконуваного файлу програми

Команди для керування мережею
1. ip - утиліта для роботи з мережею
2. ping - допоможе швидко перевірити, підключені ви до маршрутизатора або до інтернету, і дає деяке уявлення про якість зв'язку з ним
3. traceroute - це вдосконалена версія ping
